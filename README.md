# linux-dotfiles

install script dotfiles for my linux machine on WSL2

# Packages installed:

- Neovim
- Fish shell
- Fisher
- nvm
- stow
- git
- python
- tig
- yarn
- fzf
- ripgrep
- bat
- exa
- peco
- tmux
- ghq

# Install this script on a new machine

```shell
curl -o- https://raw.githubusercontent.com/cool-pants/linux-dotfiles/master/install.sh | bash
```
